# Howitzer Effects

Imported artillery impact effects used by the merged Howitzer scene.

Source pack:
- `Pixel Simulations` by `Limofeus`
- Downloaded from: `https://limofeus.itch.io/pixel-simulations`
- License noted on the asset page: `CC0 1.0`

Mapped effects:
- `explosion1/` for projectile ground impacts
- `explosion6/` for target hits
- `fireball/` for the firing animation at the muzzle
