# Apollo Effects

`explosion4/` contains the imported `Explosion4` frame sequence used by the Apollo 11 crash flash.

Source pack:
- `Pixel Simulations` by `Limofeus`
- Downloaded from: `https://limofeus.itch.io/pixel-simulations`
- Selected sequence: `Explosion4`
- License noted on the asset page: `CC0 1.0`

The runtime uses these frames only for the brief impact flash on medium and large Apollo crashes.
Crater deformation, debris motion, and ejecta timing remain scene-driven.
